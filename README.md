# App_pour_compte

## 📂 Structure du projet

```text
src/

├── 📁 pages/           # Vues principales (Home, Dashboard, Settings)
├── 📁 utils/           # Fonctions utilitaires (Formatage devises, calculs)
├── 📄 App.jsx          # Configuration des routes
└── 📄 main.jsx         # Point d'entrée React & Tailwind
